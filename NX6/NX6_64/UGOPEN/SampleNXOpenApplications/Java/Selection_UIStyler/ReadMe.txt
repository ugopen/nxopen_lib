-------------------------------------------------------------------------------
		UIStyler Selection Example
-------------------------------------------------------------------------------


Description:
------------
This example demonstrates the Selection capability of NXOpen UIStyler.
Running this example will launch a dialog which creates Edge Blend on the selected edges.

---------------------------------------
How to Build JAVA examples
---------------------------------------

Starting with NX5 you must use java version 1.5.  And set the following environmental variables.
1.set JAVA_HOME=C:\Program Files\Java\jdk1.5.0_06 
      or wherever JDK 1.5 is installed.
2.set UGII_JAVA_HOME=%JAVA_HOME%\jre
3.set PATH=%JAVA_HOME%\bin;%PATH%

Compile the example using the following command.

nmake -f Makefile_win

--------------------------------------
How to Run JAVA Examples
--------------------------------------

	1. Start NX.
	2. Open Block.prt file
	2. Go to File -> Execute ->NX Open.
	3. You will lead to window called "Execute User Function". 
	4. Navigate to the location of the jar file. 
	5. Select UIStylerSelectionExample.jar file and Click ok.
