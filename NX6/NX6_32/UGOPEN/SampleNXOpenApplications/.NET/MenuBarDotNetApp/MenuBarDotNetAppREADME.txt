-------------------------------------------------------------------------------
		MenuBarDotNet Example
-------------------------------------------------------------------------------


Description:
------------
This example demonstrates the use of the NXOpen MenuBar
namespace. It defines menubar applications with callbacks
for both the CSharp and VB.Net languages and adds the
applications to the start menu.


Please see the MenuScript chapter of the Consoldated Programmer's Guide
for a detailed description of this example, the required MenuScript
environment variables and directories, and the correct placement of
the shared library and menu files.

There are also specialized instructions for each file, 
in the comments at the top of the file.


Files included in this example:
--------------------------------
MenuBarCSharpApp.cs
MenuBarCSharpApp.men
MenuBarDotNetAppButton.men
MenuBarVbApp.men
MenuBarVbApp.vb


Note: you will need a .NET authoring license to run the example. See
the online documentation for details of licensing.
