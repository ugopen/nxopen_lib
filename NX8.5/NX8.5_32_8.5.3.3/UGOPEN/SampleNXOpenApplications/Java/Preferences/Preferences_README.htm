Preferences

This example demonstrates to set the Preferences of Sketch and Work Plane for NX session as per the requirement.Executing this example will create prefs.prt with the specified preferences. 

Files required

  1. Preferences.java
  2. Preferences_Makefile
  3. Preferences_Makefile_win

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode

  1. Compile the example with following command: 
make
  1. Start NX.
  1. In NX, go to File - Execute - NX Open, select Preferences.jar, and press Ok.
  1. This will open the new "prefs.prt" file with the specified Sketch, Grid and Workplane Preferences as mentioned in the "preferences.java" file.
  1. For cross verifying the set preferences.
      Go to Preferences->Sketch->Session Settings tab, you can observe the Name Prefixes as follows for Sketch Preferences:
        - Default Name: Prefixes
          - Sketch: SKT_
          - Vertex: VRT_
          - Line: LIN_
          - Arc: ARC_
          - Conic: CON_
          - Spline: SPL_
      Go to Preferences->Grid and Work plane, you can observe the work plane grid type is changed from rectangular to polar. Also, the following spacing preferences for grid lines are set:
        Grid and Work Plane Preferences:
          Radial Spacing-
            - Major Grid Spacing: 75.0
            - Minor Lines per Major: 3
            - Snap Points per Minor: 5
          Angular Spacing
            - Major Grid Spacing: 45.0
            - Minor Lines per Major: 4
            - Snap Points per Minor: 2
