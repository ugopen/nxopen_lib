FeatureViewer

This example demonstrates how to create a feature browser using NX Open for Java and Java Swing.  The example will create a browser using Swing that shows the features in the work part.  Clicking on a feature name in the browser will show you information about the feature.

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

  1. Compile the example
    make
  2. Start NX.
  3. Open a part.  The parts directory of the java examples contains several sample parts.  To use the sample toy car body from the sample parts directory, open ../parts/toycar_body.prt
  4. In NX, go to File - Execute - NX Open, select FeatureViewer.jar and press Ok.  This creates a browser.  Expand the Features folder of the browser to see a list of the features in the part.  Click on the name of a feature in the browser and the browser will list the parents, children, and associated expressions for the feature.
