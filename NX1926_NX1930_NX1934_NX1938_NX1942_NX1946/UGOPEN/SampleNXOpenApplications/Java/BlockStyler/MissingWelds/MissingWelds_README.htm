MissingWelds

This example demonstrates the block styler wizard. It is used to create a clearance set on an assembly and determine where the assembly is missing welds. The example also demonstrates the task navigator context menu, used to create a report of the missing welds, or export an image of a missing weld to a pdf. 

NOTE: The example is not a comprehensive tool for finding the missing welds and is solely used for demonstrating the block styler wizard block.

Files required

  1. MissingWelds.java
  2. MissingWelds.dlx : This file is located at $UGII_BASE_DIR/ugopen/SampleNXOpenApplications/C++/BlockStyler/MissingWelds.
  3. MissingWelds.prt
  4. Plate_2.prt
  5. Plate_3.prt
  6. Plate_7.prt
  7. Plate_8.prt

    8. The dlx file must be placed in one of the following locations:

    1. From where NX session is launched
    2. $UGII_USER_DIR/application
    3. For released applications, using UGII_CUSTOM_DIRECTORY_FILE is highly recommended. This variable is set to a full directory path to a file containing a list of root directories for all custom applications. e.g., UGII_CUSTOM_DIRECTORY_FILE=\ugii\menus\custom_dirs.dat

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt .

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode
  1. Compile the example with following command: 
make
  1. Start NX.
  1. Open MissingWelds.prt file.
  1. In NX, go to File - Execute - NX Open, select MissingWelds.jar, and press Ok.  The MissingWelds dialog will be displayed on the screen.
