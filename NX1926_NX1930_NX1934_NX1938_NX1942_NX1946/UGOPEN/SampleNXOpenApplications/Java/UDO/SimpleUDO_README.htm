Simple UDO

This example demonstrates each of the following callbacks:

* Display
* Selection
* Attention Point
* Fit
* Edit
* Information

Files required

  1. SimpleUDO.java

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode
  1. Compile the example with following command: 
make
  1. Start NX.
  1. In NX, go to File -> Execute -> NX Open..., select SimpleUDO.jar, and press Ok. 
    It will launch the New Screen Position dialog, which ask for screen position, Click on screen.
    It will create the new JAVA UDO Class Example.
