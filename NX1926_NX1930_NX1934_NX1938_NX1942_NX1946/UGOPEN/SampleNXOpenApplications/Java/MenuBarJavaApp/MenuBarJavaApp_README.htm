MenuBarJavaApp

This example creates a custom application and adds it to the menu bar.   Please see the "Adding Custom Applications to NX" chapter of the NX Open Programmer's Guide or search the documentation for "MenuBarJavaApp" for a detailed description of this example.

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Batch mode

Not Applicable - the MenuBar is part of the UI

Interactive mode
  1. Compile the example 
    make
  2. Copy the output MenuBarJavaApp.class file into the "application" directory in one of the following :
    1. $UGII_USER_DIR/application
    2. For released applications, using UGII_CUSTOM_DIRECTORY_FILE is highly recommended. This variable is set to a full directory path to a file containing a list of root directories for all custom applications. e.g., UGII_CUSTOM_DIRECTORY_FILE=\ugii\menus\custom_dirs.dat
  3. Copy MenuBarJavaApp.men and place it in the "application" folder with the MenuBarJavaApp.class file.
  4. In the same folder containing the "application" folder, create another folder called "startup" (if it does not already exist). Copy MenuBarJavaAppButton.men and place it in the "startup" folder.
  5. Start NX.
  6. Open any part. (The Start Menu won't appear until a part is open)
  7. In NX, go to Start - All Applications - Sample Java Application.  This will activate the custom Java application, and add a new item to the menubar titled 'Sample Java'. The buttons on the 'Sample Java' menubar application demonstrate custom buttons on the custom application.
