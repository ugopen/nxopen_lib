PMIExample

This example demonstrates the creation of a PMI dimension. 

Files required

  1. PMIExample.java
  2. PMIExample.jar
  3. PMIExample.class
  4. PMIExample_Makefile_win
  5. PMIExample_cylinder.prt : This file is located at $UGII_BASE_DIR/ugopen/SampleNXOpenApplications/C++/PMIExample.

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode
  1. Compile the example with following command: 
make
  1. Start NX.
  1. Open PMIExample_cylinder.prt part file.
  1. Enter into Modeling application through Start->All Applications->Modeling.
  1. In NX, go to File - Execute - NX Open, select PMIExample.jar, and press Ok.
  1. This will create a dimension on the cylinder in the 3D Model.
  1. Now enter into Drafting application through Start->All Applications-Drafting.
  1. In NX, go to File - Execute - NX Open, select PMIExample.jar, and press Ok.
  1. Then a dimension will be created in the drafting view.
