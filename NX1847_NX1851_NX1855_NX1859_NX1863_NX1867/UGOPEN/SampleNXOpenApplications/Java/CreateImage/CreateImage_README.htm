CreateImage

This example demonstrates the use of the NXOpen.UF (JavaWrapper of UF)API . It shows how to snag an image using NX Open for  Java Swing.Bmp and png extensions are only available for windows platform.

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

  1. Compile the example
    make
  2. Start NX.
  3. Open  any part.
  4. In NX, go to File - Execute - NX Open, select CreateImage.jar and press Ok.The results will be printed to the listing window.
  5. Click Create button. - When the image is generated, you will receive message on listing window. - By default, this image file will be created in the directory where NX was started.
