AssemblyListing

This example demonstrates how to walk through the assembly hierarchy using NX Open for Java.  For each component in the assembly, the program will list the component name, path to its part file, and the number of times the component appears in the assembly.

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Batch mode
make run

The program will list the components in the toycar_assy assembly from the parts directory of the java examples.

Interactive mode
  1. Compile the example.
    make
  2. Start NX.
  3. Open a file that contains an assembly.  A simple toy car assembly is provided in parts directory of the java examples.  To use that assembly, open ../parts/toycar_assy.prt.
  4. In NX, go to File - Execute - NX Open, select AssemblyListing.jar and press Ok.  A list of the components in the assembly will be printed in the listing window.
