 GroupUnmeshedCaeFaces.vb - Create groups of unmeshed and meshed CAE polygon faces

   Description:
   This program will create two CAE groups:

       1) Faces unrelated to elements
           - Contains all CAE polygon faces that have no related elements

       2) Faces related to elements
           - Contains all CAE polygon faces that have related elements

   It is assumed that this program will be executed in a .fem or .sim part.
 
   Instructions:
   1) Run this program from within a .fem or .sim file in NX Advanced Simulation
      - Use Tools > Journal > Play to execute GroupUnmeshedCaeFaces.vb
