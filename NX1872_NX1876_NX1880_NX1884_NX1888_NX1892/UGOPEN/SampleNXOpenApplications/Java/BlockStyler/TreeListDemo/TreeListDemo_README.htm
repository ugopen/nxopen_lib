TreeListDemo

This example demonstrates most of the functionalities of tree list such as node addition and deletion, column properties, state icons, tool tip, node edit, menu creation, drag and drop, node data usage, cross selection etc.. 

Files required

  1. TreeListDemo.java
  2. TreeListDemo.dlx : This file is located at $UGII_BASE_DIR/ugopen/SampleNXOpenApplications/C++/BlockStyler/TreeListDemo.
  3. Any part file

    4. The dlx file must be placed in one of the following locations:

    1. From where NX session is launched
    2. $UGII_USER_DIR/application
    3. For released applications, using UGII_CUSTOM_DIRECTORY_FILE is highly recommended. This variable is set to a full directory path to a file containing a list of root directories for all custom applications. e.g., UGII_CUSTOM_DIRECTORY_FILE=\ugii\menus\custom_dirs.dat

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode
  1. Compile the example with following command: 
make
  1. Start NX.
  1. Open any part file with at least one feature in it.
  1. In NX, go to File - Execute - NX Open, select TreeListDemo.jar, and press Ok.  The Tree List Demonstration dialog will be displayed on the screen.
  1. It is advisable to expand the dialog-width to see the complete group title, labels etc which help in understanding the tree.
  1. Observed that the tree functionality is grouped and each group caters to particular functionality of tree. The group title would be helpful in identifying particular functionality.
  1. In some groups instructions are provided. Follow the same to understand the tree list behavior.
