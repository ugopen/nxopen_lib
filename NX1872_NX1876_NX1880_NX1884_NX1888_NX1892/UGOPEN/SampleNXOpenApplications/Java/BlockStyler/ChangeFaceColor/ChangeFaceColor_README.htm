ChangeFaceColor

This example has a simple dialog with a Face Collector block and an Object Color Picker block. The user can select faces that should have the color changed using the Face Collector and see pick the new face color using the Object Color Picker. Clicking on OK or Apply will set the color of the selected faces to the selected color. The user can pick different selection intent rules to use for selecting faces using the Face Rule list on the Top Border Bar in NX. 

The example uses NX Open Java API methods and properties to access the faces and initialize the dialog elements. The Block Styler dialog elements are controlled using methods on the Block Styler classes instead of using the generic property list methods. 

Files required

  1. Any part file
  2. ChangeFaceColor.java
  3. ChangeFaceColor.dlx

    4. The dlx file must be placed in one of the following locations:

    1. From where NX session is launched
    2. $UGII_USER_DIR/application
    3. For released applications, using UGII_CUSTOM_DIRECTORY_FILE is highly recommended. This variable is set to a full directory path to a file containing a list of root directories for all custom applications. e.g., UGII_CUSTOM_DIRECTORY_FILE=\ugii\menus\custom_dirs.dat

How to run the example

Unless noted otherwise, the commands in these instructions must be run from an NX command prompt.  See the general notes of the examples for instructions on how to start an NX command prompt.

Instructions for Windows: All commands are the same as in Unix except substitute "make" with "nmake -f Makefile_win".

Interactive mode
  1. Compile the example with following command: 
make

  1. Start NX.

  2. Open any part file with some faces that you can select to change the face color. You can also create a new part file and create some faces by inserting features that create solid geometry.

  3. In NX, go to File - Execute - NX Open, select ChangeFaceColor.jar, and press Ok.  The Change Face Color dialog will be displayed on the screen.

  4. Select one or more faces in the Face Collector block. You can specify face selection intent rules from the Face Rule list on the NX Top Border Bar.

  5. Select the new color for your selected faces using the Object Color Picker block.

  6. Click OK or Apply to set the color of the selected faces to the new color.
